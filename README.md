# Restaurant Recommendation Dialog System

## Environment Setup
```shell
pip install pipenv
pipenv install
```

## Run
```shell
python main.py
```

usage: main.py [-h] [--dataset_path DATASET_PATH]

  --dataset_path DATASET_PATH
                        Path to dialog classification dataset
